# GeoTests

Um pacote Julia para testes geoespaciais.

## Instalação

Para instalar o pacote, execute o seguinte comando no REPL do Julia:

```julia
using Pkg
Pkg.add(url="https://github.com/seu-usuario/GeoTests.jl")
```
